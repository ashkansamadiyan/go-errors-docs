Welcome to the **go-errors** documentation! This library brings Go-style error handling to JavaScript/TypeScript, providing a type-safe and elegant way to handle errors without try-catch blocks.

## Quick Start

Install the library using bun:

```bash
bun add go-errors
```

Basic usage:

```typescript
import { goSync, goFetch } from 'go-errors';

// Synchronous operations
let [value, err] = goSync(() => {
  if (Math.random() > 0.5) throw new Error('Bad luck!');
  return 42;
});

if (err) {
  console.error('Something went wrong:', err.message);
} else {
  console.log('Got value:', value);
}

// HTTP requests with type safety
interface User {
  id: number;
  firstName: string;
  lastName: string;
  fullName: string;
}

let [user, err] = await goFetch<User>('/api/user/123', {
  responseTransformer: (data) => ({
    ...data,
    fullName: `${data.firstName} ${data.lastName}`
  })
});

if (err) {
  console.error('Failed to fetch user:', err.message);
} else {
  console.log('User:', user.fullName);
}
```
